This project repository is connected to the journal article ‘Pesticides Never Walk Alone: Validating Mixture Toxicity and Bioaccumulation Models in Soil Arthropods ’ by Wehrli et al. 2026 (doi TBA). 

The key element of this repository is the integration of a one-compartment TK model including phase-one biotransformation (TK-metabolism) and dynamic exposure in the BYOM platform (https://www.debtox.info/byom.html). 

How to run this model for your own work?    
Navigate to "BYOM/TK-metabolism", run "pathdefine.m", use "byom_metabolism_template.m" as your template.
